# 建材小程序

这是一个微信小程序项目，包含首页、分类、购物车、商品详情和我的页面，以及自定义导航栏组件。

## 目录结构

- components/：公共组件
  - navbar/：自定义导航栏
- pages/：页面
  - index/：首页
  - category/：分类页
  - cart/：购物车页
  - goods-detail/：商品详情页
  - mine/：我的页
- images/：图片资源

## 运行方式

1. 使用微信开发者工具导入本项目目录。
2. 选择小程序项目类型并填写 AppID（或使用测试号）。
3. 点击“编译”即可预览。

## 说明

- 页面配置位于各页面目录的 `.json` 文件中。
- 全局配置位于 `app.json`。
